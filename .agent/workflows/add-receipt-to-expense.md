To add a receipt to an expense , you’ll need to upload the receipt when you create or edit an expense entry. Here’s a step-by-step procedure:

# How to Add a Receipt to an Expense in Hubstaff

1. **Log in to Hubstaff**
Go to the Hubstaff web app and sign in with your credentials.

2. **Navigate to Expenses**
From the main dashboard, select the organization (if you have more than one).
Click on the “Expenses” tab in the sidebar menu.

3. **Add a New Expense**
Click the “Add expense” or “+ Add” button.

4. **Fill Out Expense Details**
Enter the required details such as:
- Date of the expense
- Amount
- Category (e.g., Travel, Meals)
- Project (if applicable)
- Description or notes

5. **Upload the Receipt**
Look for an option labeled “Attach receipt,” “Upload receipt,” or a paperclip icon.
Click it and select the receipt file from your computer or device (usually accepts images like JPG, PNG, or PDF files).

6. **Save the Expense**
After attaching the receipt and filling out all details, click “Save” or “Submit.”

7. **Review**
The expense will now appear in the Expenses report, and the receipt will be accessible via the “Receipt” column.

## Editing an Existing Expense:
If you need to add a receipt to an existing expense, find the expense in the list, click “Edit,” and follow steps 5–7 above.

## Tips:
- Make sure the receipt image or file is clear and legible.
- You can usually only upload one receipt per expense entry.
- If you need help with a specific step or run into any issues uploading receipts, let me know!
